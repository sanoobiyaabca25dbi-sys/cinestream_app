 🎬 CineStream Content Analytics Dashboard

An interactive **Streamlit dashboard** developed to analyze the performance of CineStream OTT content. This project transforms raw streaming data into meaningful business insights using interactive filters, KPI cards, and data visualizations.

---

## 🌐 Live Demo

https://cinestreamapp-ubjyafvnbz5g7qyk2yl9p4.streamlit.app/

# 📖 Project Overview

The **CineStream Content Analytics Dashboard** helps users explore and analyze OTT content performance across multiple dimensions such as content type, genre, language, revenue, ratings, and audience engagement.

The dashboard enables users to interactively filter the dataset and gain valuable insights through charts, KPIs, and smart status indicators.

---

# ✨ Key Features

### 📊 Interactive Dashboard
- Clean and user-friendly interface
- Responsive layout with multiple tabs
- Interactive sidebar filters

### 📈 Key Performance Indicators (KPIs)
- Total Titles
- Total Views
- Total Watch Hours
- Average IMDb Score

### 🎛️ Interactive Filters
- Genre
- Language
- Content Type
- Age Rating
- IMDb Score Range
- Runtime Range
- Added Date Range

### 📉 Visualizations
- Monthly Content Release Trend
- Content Type Distribution
- Top 10 Genres by Views
- Language → Genre Treemap
- Production Cost vs Revenue Analysis
- Average ROI by Genre
- IMDb Score Distribution
- IMDb Score vs Views Correlation

### 📥 Additional Features
- Download filtered dataset as CSV
- Smart status messages
- Cached data loading for better performance
- Interactive filtering with Streamlit Forms

# 📂 Dataset Information

The dataset contains approximately **520 OTT titles** with information about:

- Content ID
- Title
- Content Type
- Genre
- Language
- Country
- Director
- Release Year
- Runtime
- IMDb Score
- Views
- Watch Hours
- Revenue
- Production Cost
- Subscribers Gained
- Trending Status
- Critic Rating

### Derived Columns

- Profit_Cr
- ROI_Pct
- Performance_Band
